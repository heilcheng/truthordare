# Truth or Dare (Cantonese)
Website: https://heilcheng.github.io/truthordare

## Features
- All / Truth / Dare
- Generate a question with one click
- Able to add your own questions (in the browser)

## Local Setup (Frontend)
```bash
npm install
npm start
```

## Local Setup (Backend, Optional)
```bash
cd server
npm install
npm start
```
